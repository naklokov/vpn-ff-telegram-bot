Инструкция Windows

1. Копируете длинную строку подключения присланную выше
2. Скачиваете клиент Hiddify (https://github.com/hiddify/hiddify-next/releases/latest/download/Hiddify-Windows-Setup-x64.exe)
3. Устанавливаете его
4. Открываете приложение Hiddify
5. Нажимаете пункт добавить (+)
6. Вставляете текст скопированный ранее
7. Готово
